Learning List

HTML
HTML là viết tắt của cụm từ HyperText Markup Language (Ngôn ngữ đánh dấu siêu văn bản).
CSS
CSS là viết tắt của cụm từ Cascading Style Sheet.
Là một ngôn ngữ quy định các hiển thị của các thẻ HTML trên trang web.
Javascript
Javascript là một ngôn ngữ lập trình kịch bản dựa trên đối tượng được phát triển từ các ý niệm nguyên mẫu.
